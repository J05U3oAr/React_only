# Juego Snake

Juego Snake realizado en base a React, en donde toda la estructura de este mismo se encuentra dentro del index.html.


## Instrucciones

W = Se utiliza para ir hacia arria
S = Se utiliza para ir hacia abajo
D = Se utiliza para ir hacia a la derecha
A = Se utiliza para ir a la izquierda

Estos son los controles para poder llevar a cabo el juego, aunque además de estas también se pueden utilizar las flechas del teclado. 

## Tecnologías 

- HTML 5
- CSS3
- JavaScript
- React
- React 18 — biblioteca principal para construir la interfaz con componentes
- ReactDOM 18 — para montar la app en el DOM
- Babel Standalone — transpila JSX a JavaScript que el navegador entiende
